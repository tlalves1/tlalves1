## 🏠 Meu Homelab & Projetos Pessoais

Para testar novas tecnologias, estudar conceitos de redes e simular ambientes de infraestrutura, eu mantenho um **servidor caseiro (Homelab)** pessoal. Nele, utilizo o **CasaOS** como sistema de gerenciamento e hospedo os seguintes serviços:

* 🐳 **CasaOS / Docker:** Base do servidor para orquestração simplificada dos serviços em containers.
* 🛡️ **AdGuard Home:** Meu servidor de DNS local e privado, configurado para bloqueio de anúncios e controle de rede.
* ☁️ **Nextcloud:** Servidor de nuvem privada para armazenamento, sincronização de arquivos e produtividade.
* 🎬 **Jellyfin & Navidrome:** Servidores de mídia pessoais para streaming de vídeos e músicas na rede doméstica.
* 📂 **File Server:** Servidor de arquivos centralizado para backup e compartilhamento de dados local.
